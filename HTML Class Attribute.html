HTML Class Attribute

      The HTML class attribugte is used to specify a class for an HTML element.
      Multipule HTML element can share the same class.

Using The class Attribute

      The class attribute is often used to point to a class name in style sheet. It can also used by a JavaScript to accss and manipulate elements with the specific
      class name.
